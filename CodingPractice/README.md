# 코딩 연습 - 구조체

교안의 예제 코드를 직접 따라 작성하고 실행하는 프로젝트입니다.

> 각 과제의 코드를 `Program.cs`에 작성하세요.
> 이전 과제의 코드가 남아있으면 구조체 이름이 겹쳐서 오류가 발생할 수 있습니다. 이전 과제의 코드를 지우거나 주석 처리한 후 다음 과제를 진행하세요.

---

## 1. 구조체 만들기

### 1-1. 기본 구조체 선언

2차원 좌표를 표현하는 `Point` 구조체를 정의하세요. `X`, `Y` 필드를 가지며, 구조체 변수를 선언한 뒤 필드에 값을 할당하고 출력합니다.

**실행 결과**

```
X: 100, Y: 200
```

### 1-2. 다양한 타입의 필드를 가진 구조체

상품 정보를 담는 `Product` 구조체를 정의하세요. `Id` (int), `Title` (string), `Price` (decimal) 필드를 가지며, 구조체 변수에 값을 할당하고 출력합니다.

**실행 결과**

```
번호: 1
상품명: 좋은 책
가격: 10000
```

### 1-3. 명함 정보 구조체

명함 정보를 담는 `BusinessCard` 구조체를 정의하세요. `Name` (string), `Age` (int), `Address` (string) 필드를 가지며, 구조체 변수에 값을 할당하고 출력합니다.

**실행 결과**

```
홍길동, 21, 서울시
```

---

## 2. 생성자를 가진 구조체

### 2-1. 구조체 생성자

`Point` 구조체에 두 좌표를 받아 초기화하는 생성자를 정의하세요. `new Point(100, 200)`으로 초기화한 뒤 출력합니다.

**실행 결과**

```
X: 100, Y: 200
```

---

## 3. 구조체 배열

### 3-1. 구조체 배열 사용

`BusinessCard` 구조체 배열을 선언하고 3명의 데이터를 저장하세요. 반복문으로 전체 데이터를 출력합니다.

**실행 결과**

```
홍길동은(는) 17살입니다.
백두산은(는) 102살입니다.
임꺽정은(는) 31살입니다.
```

### 3-2. 성적 관리 프로그램

`Score` 구조체를 정의하세요. `Name`, `Kor`, `Eng`, `Total`, `Average` 필드를 가지며, 3명의 성적 데이터를 배열로 관리하고 총점과 평균을 계산하여 출력합니다.

**실행 결과**

```
이름	총점	평균
홍길동	190	95
백두산	170	85
임꺽정	150	75
```

## 4. 구조체 매개변수

### 4-1. 구조체를 함수에 전달하기

`Member` 구조체를 정의하고, 개별 매개변수를 사용하는 함수와 구조체 매개변수를 사용하는 함수를 각각 작성하세요. 두 방식으로 데이터를 출력합니다.

**실행 결과**

```
이름: 홍길동, 나이: 21
이름: 백두산, 나이: 100
```

---

## 5. 중첩 구조체

### 5-1. 구조체 안에 구조체

`Score` 구조체(`Kor`, `Eng` 필드)와 `Student` 구조체(`Num`, `Name`, `Scores` 필드)를 정의하세요. `Student` 구조체의 `Scores` 필드는 `Score` 타입입니다. 3명의 학생 데이터를 배열로 관리하고 출력합니다.

**실행 결과**

```
1 - 홍길동 : 100, 90
2 - 백두산 : 90, 80
3 - 임꺽정 : 80, 70
```

## 6. Char 구조체 활용

### 6-1. 문자 종류 확인과 변환

`char.IsUpper()`, `char.IsLower()`, `char.IsDigit()`, `char.IsWhiteSpace()` 메서드를 사용하여 문자의 종류를 확인하세요. `char.ToLower()`, `char.ToUpper()`로 대소문자를 변환합니다.

**실행 결과**

```
A은(는) 대문자
a은(는) 소문자
1은(는) 숫자
공백 문자
a
A
```

## 7. DateTime 구조체 활용

### 7-1. 날짜와 시간 다루기

`DateTime.Now`로 현재 날짜를 가져오고, `new DateTime()`으로 특정 날짜를 생성하세요. `AddDays()`로 날짜를 더하고, 두 날짜 사이의 간격을 계산합니다.

**실행 결과**

```
현재: 2024-12-25 오후 3:30:00
생일: 2000년 05월 15일
일주일 후: 2025-01-01
나이: 24세
```

---

## 8. 값 형식 vs 참조 형식

### 8-1. 구조체의 값 복사

`PointStruct` 구조체를 정의하고, 구조체 변수를 다른 변수에 대입하세요. 복사된 변수의 값을 변경해도 원본에 영향이 없음을 확인합니다.

**실행 결과**

```
p1.X = 10
p2.X = 20
```
